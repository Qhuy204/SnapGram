# SnapGram

SnapGram turns diagram screenshots into editable Draw.io files.

## Stack

- Next.js App Router frontend
- Next.js route handlers as `/api/*` proxies
- Python backend for image parsing and Draw.io export
- OpenAI Responses API with JSON Schema structured output when `OPENAI_API_KEY` is set

## Run

Install frontend dependencies:

```powershell
npm install
```

Start the Python backend:

```powershell
python backend/server.py
```

Start Next.js:

```powershell
$env:PY_BACKEND_URL="http://127.0.0.1:8000"
npm run dev
```

Open `http://localhost:3000`.

Without `OPENAI_API_KEY`, the Python backend returns the built-in demo graph so the UI and Draw.io export still work. With a key, upload an image and the backend calls the OpenAI Responses API for diagram JSON only; XML is generated by Python.
